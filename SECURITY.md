# Security Policy

## Supported Versions
Security updates are applied only to the latest major release of **Votha Sentient Overlay**.

| Version | Supported |
|----------|------------|
| Latest   | ✅ |
| Older    | ❌ |

---

## Reporting a Vulnerability

If you discover a security vulnerability, **please report it privately**.

- Email: **dwenrur@gmail.com**  
- Subject line: `Vulnerability Report – Votha Sentient Overlay`  
- Include:  
  - A detailed description of the issue  
  - Steps to reproduce  
  - Affected versions or configurations  
  - Suggested fixes (if any)

Do **not** open a public GitHub issue for security vulnerabilities.

---

## Disclosure Policy
- We will acknowledge receipt of your report within **48 hours**.  
- A fix or mitigation plan will typically be shared within **7 days**.  
- Once resolved, we’ll issue a release and credit you (if desired).

---

## Responsible Research Guidelines
Please:
- Avoid public disclosure until we’ve deployed a fix.  
- Do not attempt to exploit or test vulnerabilities on live systems.  
- Do not use automated scanning tools that may disrupt services.

Thank you for helping keep **Votha Sentient Overlay** safe for everyone.
